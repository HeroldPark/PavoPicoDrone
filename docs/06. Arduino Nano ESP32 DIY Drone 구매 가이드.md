## ⚠️ 주의사항

### 안전 수칙
- 배터리 충전 시 안전한 장소에서
- 첫 비행은 넓은 공간에서
- 프로펠러 가드 필수 착용

### 기술적 고려사항
- Arduino 펌웨어 개발 시간 충분히 확보
- PID 튜닝용 시뮬레이터 활용
- 백업 부품 (프레임, 프로펠러) 추가 구매 권장

**총 예상 시간**: 조립 1-2주, 펌웨어 개발 2-4주, 튜닝 2-3주
