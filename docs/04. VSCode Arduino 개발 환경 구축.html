⚙️ 4단계: Arduino Nano ESP32 보드 설정

1

ESP32 보드 패키지 설치

Ctrl+Shift+P → Arduino: Board Manager

보드 패키지 URL 추가
https://raw.githubusercontent.com/espressif/arduino-esp32/gh-pages/package_esp32_index.json
  1. Arduino IDE → 파일 → 환경설정
  2. "추가 보드 매니저 URLs"에 위 URL 추가
  3. 도구 → 보드 → 보드 매니저
  4. "esp32" 검색 → "esp32 by Espressif Systems" 설치
2

보드 및 포트 선택

Ctrl+Shift+P → Arduino: Board Config

  • 보드: Arduino Nano ESP32
  • 포트: COM3 (Windows) 또는 /dev/ttyUSB0 (Linux)
  • 업로드 속도: 921600
  • CPU 주파수: 240MHz
💡 포트 찾기: 장치 관리자(Windows) 또는 ls /dev/tty*(Linux/Mac)에서 확인
3

🚨 7단계: 문제 해결 가이드

❌ 자주 발생하는 문제들

1. 컴파일 오류

  • 헤더 파일 없음: includePath 확인
  • 라이브러리 없음: Library Manager에서 설치
  • 보드 정의 없음: 보드 패키지 재설치
일반적인 컴파일 오류 해결
# 캐시 정리 arduino-cli cache clean # 라이브러리 다시 설치 arduino-cli lib install "ESP32Servo" arduino-cli lib install "MPU9250_asukiaaa" # 보드 재설치 arduino-cli core update-index arduino-cli core install esp32:esp32

2. 업로드 실패

  • 포트 오류: 다른 프로그램에서 포트 사용 중
  • 드라이버 문제: CP210x 또는 CH340 드라이버 설치
  • 권한 문제: 관리자 권한으로 실행

3. IntelliSense 작동 안함

  • 경로 설정: c_cpp_properties.json 확인
  • 컴파일러 경로: 올바른 gcc 경로 설정
  • 캐시 문제: VSCode 재시작
