⚙️ 조종기 설정 가이드

🎮 RadioMaster TX16S 설정 단계

1. 기본 펌웨어 설정

  • EdgeTX 업데이트: 최신 버전으로 업그레이드
  • 언어 설정: System → 언어 → 한국어
  • RF 프로토콜: FrSky D16 또는 ExpressLRS 선택

2. 드론 모델 생성

  • 모델 추가: Model → [+] → "Pavo Pico"
  • 모델 타입: Airplane → Quadcopter
  • RF 모듈: Internal → FrSky D16 (EU LBT)

3. 채널 매핑 설정

📋 표준 채널 매핑

  • CH1 (A): Roll → 우측 스틱 좌우
  • CH2 (E): Pitch → 우측 스틱 상하
  • CH3 (T): Throttle → 좌측 스틱 상하
  • CH4 (R): Yaw → 좌측 스틱 좌우
  • CH5 (SA): ARM/DISARM → SA 스위치
  • CH6 (SB): Flight Mode → SB 스위치

4. 스위치 및 모드 설정

  • ARM 스위치: SA ↑ = DISARM, SA ↓ = ARM
  • 모드 스위치: SB ↑ = STABILIZE, SB ↓ = ACRO
  • 페일세이프: 모든 채널 Hold, Throttle → 900μs

🥽 Walksnail Goggles X 설정

1. 초기 설정

  • 언어 설정: Settings → Language → 한국어
  • 채널 검색: Auto Scan으로 VTX 검색
  • 화질 설정: 1080p60 고정

2. Avatar HD V2 바인딩

  • 바인딩 모드: 고글과 VTX 동시에 바인딩 버튼
  • 채널 확인: 연결된 채널 번호 기억
  • 신호 품질: RSSI 값 확인

3. OSD 설정

  • 배터리 전압: 좌상단에 표시
  • 비행 시간: 우상단에 표시
  • 인공 수평선: 중앙에 표시
  • GPS 정보: 하단에 표시 (추후)

💡 구매 팁 & 주의사항

⚠️ 구매 전 확인사항

  • 주파수 대역: 한국은 2.4GHz 전용 (5.8GHz는 불법)
  • 전파인증: KC 인증 제품 또는 실험국 신고 필요
  • 배터리 규정: 리튬 배터리 운송 규정 확인
  • A/S 지원: 국내 대리점 또는 커뮤니티 지원 확인

🛒 추천 구매처

  • 해외직구: Banggood, GetFPV, RDQ (저렴하지만 배송 느림)
  • 국내업체: 코스모테크, 멀티로터코리아 (빠른 배송, A/S)
  • 중고거래: 네이버 카페, 당근마켓 (가성비 좋음)
  • 공식대리점: 보증 및 지원 최고
